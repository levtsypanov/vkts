# vk-ts
VK Mini Core shared JS libs


## Установка

**github**:

```shell
npm i github:levtsypanov/vkts --save
```

**mvk**:

```shell
npm i gitlab.mvk.com:mini-core/vkts
```

## Использование

```jsx
import { getUserToken } from '@mini-core/vkts/dist/bridge/method';
```
или

```jsx
import { getUserToken } from "@mini-core/vkts";
```
